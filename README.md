# Security Analysis Suite

Программный комплекс для автоматизированного аудита безопасности программного обеспечения.

### Особенности проекта:
- **Быстрый поиск:** Интеграция с Vulners API для моментальной проверки версий ПО на наличие CVE.
- **Пакетный анализ:** Автоматизированный аудит списков программного обеспечения.
- **AI-Интерпретатор:** Интеграция с Google Gemini для анализа и упрощенного объяснения сложных отчетов об уязвимостях.

### Автор: Ulan
Этот проект был разработан в рамках учебной практики для автоматизации задач по информационной безопасности.

### Технологический стек:
- Python 3.11+
- Streamlit (интерфейс)
- Vulners API (база уязвимостей)
- Google Generative AI (ИИ-аналитика)
